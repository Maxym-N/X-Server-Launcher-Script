### Installation

To install the script system-wide so you can run it simply by typing `startxx` from anywhere:

```bash
sudo curl -L -o /usr/local/bin/startxx https://github.com/Maxym-N/X-Server-Launcher-Script/main/startxx
sudo chmod +x /usr/local/bin/startxx

After installation, run the script with:

startxx
